# E-Commerce Sales Analysis Dashboard

This project provides a small interactive dashboard built with [Streamlit](https://streamlit.io/) for exploring e-commerce datasets. Users can upload daily, monthly or yearly sales data in CSV format and generate quick visualizations.

## Features
- Upload any CSV file containing date and numeric columns
- Aggregate sales by day, month or year
- Interactive line chart using Altair

## Setup
Install the required packages:
```bash
pip install streamlit pandas altair
```

## Usage
Launch the dashboard with:
```bash
streamlit run dashboard.py
```
Follow the link printed in the terminal to open the web UI and upload your data.
